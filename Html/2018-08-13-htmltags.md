This is html tags which I learned until now.

```html
<br> 닫음 없음. 단순한 new line
<p> </p> 단락 설정. 정보로서의 가치를 더함
<img> 이미지를 삽입 가능하게 함. attribute로 src, width등을 설정 가능
<li> </li> 리스트. <ul>과 <ol>을 부모로 가짐
<ul> </ul> unordered list. li의 부모태그. 그루핑
<ol> </ol> ordered list. li의 부모태그. 자동으로 넘버링
<title> </title> 웹 페이지의 제목 설정. 책 표지와 같은 역할
<head> </head> 본문 설명. 설정들을 모아둠
<body> </body> 본문
<meta> 닫음 없음. 브라우저가 어떤 규약으로 파일을 열것인지 정함.
<html> </html> 최상위 태그
<!doctype html> 맨 위에 써줌
<a> 태그의 제왕. 링크를 표현. attribute로 href, target, title등을 설정 가능. href는 링크, target은 새창에 열 것인지, title은 마우스를 올렸을 때 설명을 띄워줌.
```
